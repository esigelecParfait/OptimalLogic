1

Créer les clés Google Cloud ⏱ 10 min

Cliquez sur chaque lien — ils ouvrent directement la bonne page.

1

Ouvrir Google Cloud Console

🌐console.cloud.google.com/projectcreate

Connectez-vous avec le compte Google qui gère les fiches Business de vos clients.
Nom du projet : tapez OptimalLogic → Créer

2

Activer les 2 APIs (lien direct)

Cliquez les 2 liens ci-dessous (s'ouvrent dans de nouveaux onglets) → bouton Activer sur chacun :

🔗API Business Profile Performance
🔗API My Business Account Management
3

Configurer l'écran de consentement OAuth

🔗console.cloud.google.com/apis/credentials/consent

Choisir Externe → Créer
Remplir uniquement : Nom de l'application = OptimalLogic + votre email.
Cliquer Enregistrer et continuer sur les 3 écrans suivants sans rien changer.

4

Créer les identifiants OAuth 2.0

🔗console.cloud.google.com/apis/credentials/oauthclient
ChampValeur à saisir
Type d'applicationApplication Web
NomOptimalLogic Server
URI de redirection autoriséhttps://VOTRE-SITE.vercel.app/api/admin/google-auth/callback

Cliquer Créer → Copier et noter le Client ID et le Client Secret.

⚠️ Copiez ces deux valeurs maintenant dans un fichier texte — vous en aurez besoin à l'étape suivante.
2

Ajouter les variables dans Vercel ⏱ 2 min

Settings → Environment Variables → ajouter ces 3 lignes

Variable Valeur Source
GOOGLE_CLIENT_ID Le Client ID copié à l'étape 1.4 Google Cloud Console
GOOGLE_CLIENT_SECRET Le Client Secret copié à l'étape 1.4 Google Cloud Console
NEXT_PUBLIC_APP_URL https://votre-site.vercel.app URL de votre déploiement
⚠️ Cochez "All Environments" pour chaque variable. Puis dans l'onglet Deployments → bouton Redeploy → attendez le point vert.

Alternative CLI (plus rapide) :

vercel env add GOOGLE_CLIENT_ID
vercel env add GOOGLE_CLIENT_SECRET
vercel env add NEXT_PUBLIC_APP_URL
vercel --prod # redéploiement
3

Autoriser en 1 clic ⏱ 1 min

Une seule fois — donne accès permanent à vos fiches Google

1

Ouvrir ce lien dans votre navigateur

🔗 https://VOTRE-SITE.vercel.app/api/admin/google-auth?secret=VOTRE_ADMIN_SECRET

→ Sélectionnez le compte Google qui gère les fiches Business → Autoriser

ℹ️ Si Google affiche "application non vérifiée" → cliquez Paramètres avancés → Accéder à OptimalLogic. C'est normal pour une application privée.
2

Copier le Refresh Token et le coller dans Vercel

La page affiche un code violet — copiez-le et ajoutez-le dans Vercel :

VariableValeur
GOOGLE_REFRESH_TOKENLe code affiché sur la page

→ Redéployez à nouveau (Vercel → Deployments → Redeploy)

🔒 Ne partagez jamais ce token. Il donne accès à toutes vos fiches Google Business.
4

Lier tout automatiquement ⏱ 2 min

La page détecte les correspondances et lie tout en un clic

1

Ouvrir la page d'administration

🔗 https://VOTRE-SITE.vercel.app/admin/google-setup?secret=VOTRE_ADMIN_SECRET

La page se charge et récupère automatiquement toutes vos fiches Google Business.

2

L'algorithme détecte les correspondances

Les menus déroulants se pré-remplissent automatiquement en comparant les noms des fiches Google avec les noms des clients dans Supabase.

Les correspondances fiables sont marquées Auto ✓. Vérifiez les autres et corrigez si nécessaire.

3

Cliquer "⚡ Lier tout en un clic"

Toutes les liaisons détectées sont enregistrées dans Supabase en quelques secondes. Pas besoin de copier des identifiants ou de toucher à Supabase manuellement.

✅ C'est terminé. Les métriques Google (vues, clics, avis, note) se mettront à jour automatiquement chaque lundi à 8h30.
🔗

Problèmes fréquents

SymptômeCauseSolution rapide
"No refresh_token" Application déjà autorisée myaccount.google.com/permissions → supprimer OptimalLogic → relancer étape 3
Erreur 403 sur l'API APIs non activées Vérifier que les 2 APIs sont bien activées (bouton "Gérer" visible)
Page admin : aucune fiche Mauvais compte Google Relancer étape 3 avec le compte qui gère les fiches Business
Métriques à zéro Semaine sans activité Normal — Google remonte 0 si pas de vues/clics la semaine passée
"Non autorisé" sur /api/admin Mauvais ADMIN_SECRET Vérifier la variable ADMIN_SECRET dans Vercel (copier exactement)
